Développement Web

Définitions

La page Web est obtenue par interprétation d'un code HTML envoyé par le serveru en réponse à la requête qui lui est envoyée ( adresse en http://www.etc ). C'est la navigateur qui interpréte ce code.

Il existe 2 sortes de balises :

  1. les balises les plus courantes encadrent du code qui se termine par une balise dite "fermante". Exemple : <balise>contenu HTML</balise>
  2. les balises dites orphelines qui n'encadrent aucun code et qui se ferment toutes seules. Exemple :<br/> <hr/> <img/> <input/> etc...
